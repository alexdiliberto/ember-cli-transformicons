Compatibility
------------------------------------------------------------------------------

* Ember.js v2.18 or above
* Ember CLI v2.13 or above
* Node.js v8 or above

Installation
------------------------------------------------------------------------------

```sh
ember install ember-transformicons
```


Usage
------------------------------------------------------------------------------

Simply use the provided Transformicon components in any of your Ember applications
